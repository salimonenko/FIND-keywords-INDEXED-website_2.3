# FIND-keywords-INDEXED-website_2.3
Improved version of FIND-keywords-INDEXED-website_2.1
This version significantly reduces fuzzy search time. A minor refactoring has also been implemented. The total time for a fuzzy search (taking into account different forms of search words) for files containing keywords is approximately 0.2 seconds (for approximately 2000 files with a total size of approximately 30 MB).
The programs have been tested in PHP 5.3. They haven't yet been tested in newer versions of PHP. They should probably work as well.


Улучшенная версия FIND-keywords-INDEXED-website_2.1
В этой версии существенно снижено время нечеткого поиска. Также сделан небольшой рефакторинг. Общее время нечеткого (с учетом разных форм искомых слов) поиска файлов, содержащих ключевые слова, составляет примерно 0,2 секунды (для примерно 2000 файлов общим объемом около 30 МБ). 
Работа программ проверена в PHP 5.3. В новых версиях PHP пока не проверялось. Вероятно, тоже должно все работать.
